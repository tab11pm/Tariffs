## Подготовка окружения

- Создайте `.env` файл
- Добавте туда `SERVER_URL="https://example.com"`
- Сохраните файл

## Запуск при помощи Docker (самий быстрий способ)

- Откройте терминал
- Убедитесть что вы находитес в папке проекта
- В терминале надо ввести эту команду `docker compose up -d`

### Ошибки

- Убедитесь что `docker` установлен
- Для того чтобы проверить ведите в терминале `docker -v`
- Если есть версия то значить вы не настроили окружения проверте файли `Dockerfile` и `docker-compose.yml`

## Запуск при помощи Node.js

- Для начала убедитесть что `Node.js` и `npm` установлен
- Ведите в терминале

```bash
	node -v
	npm -v
```

- После этого ведите `npm i`
- Для запуска приложения `npm run dev`

Откройте браузер и ведите http://localhost:3000. Поздравляю вы запустили проект
